Choosing 'Edit Entrainer Program' from the file menu brings up the following selection dialog:

Entrainer Program Chooser

Choose the Entrainer program you wish to edit. By default the dialog opens in the 'xml' subdirectory of the Entrainer install.

Once the file has been selected, the XML Editor is displayed:

Entrainer Program Editor

*Add icon by http://dryicons.com

**Intervals menu new in 1.0.1 Final

Each tab represents a unit of time specified by the minutes and seconds. During this time the values of the fields will vary from the 'Start' values to the 'End' values. The start and end settings can be tested by pressing either 'Test' button, and the values can be changed while testing. To test a new value, type it into the field and either hit the enter key or tab out of the field:

*Add icon by http://dryicons.com

Displaying the Unit Chart

Clicking the 'Show Chart' button in the editor displays the chart for the xml program. Hovering over any of the nodes will display its value. To zoom in, click and drag the mouse from the top left corner of the area you wish to view to the bottom right. To revert to the original view, click, drag and release the mouse in the opposite direction (right to left).

New Entrainer Program

Choosing 'New Entrainer Program' from the file menu brings up the editor in new file mode. Add as many units as you wish to create your custom program, then click 'Save' to save the program.

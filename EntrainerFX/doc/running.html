Executing

Double click the EntrainerFX.jar file in the Entrainer installation directory, or start the application as per the instructions on the Index page. After the splash screen disappears, the application starts up.

Controls

The following describes the controls that are available to the user.

Background Control (New in 5.1.0)

As the last major new feature prior to full JavaFX conversion I've implemented full control of the background images or colours. Should the 'Dynamic' radio button be pressed a single click of the image directory field allows the user to choose any directory containing image files (jpeg, bmp, gif & png files). The directory is recursively scanned and the images are randomly changed using the duration and transition values selected.

Should the 'Static' radio button be selected the user is able to single click the static image field and choose a single picture for display. Should the 'Lock Image' checkbox be selected the image can only be changed by clicking the static image field. If the checkbox is not selected a new image can be chosen when the 'Dynamic' radio button is selected and the background is cycling through random images - see an image you like, then click the 'Static' radio button before it transitions to the next image.

Should the 'No Image' radio button be selected the user is able to choose a static colour for the background via the colour picker control, or choose random background colours during play by selecting the 'Psychedelic' checkbox.

In all 3 modes selecting the 'Flash Background' checkbox will cause the background to flash in time with the chosen entrainment frequency

Shimmer Effect (New in 1.1.5 Final)

The user can choose to place a shimmer effect over top of the application while it is running. Colours and transparency values gently shift over time, providing a hypnotic oil-on-water effect.




The Interval Menu (New in 1.0.1 Final)

Intervals can now be added to the base frequency. The formula for the addition is as follows:

  • Interval Frequency = Base Frequency + (Base Frequency * Interval) where Interval is resolved from the chosen fraction.

    • The included intervals are the chromatic scale (1/12 thru 12/12) and the Golden Ratios (-8/21 and 13/21). Users can add custom intervals as desired, and all intervals are available for use in Entrainer programs.

    The interval menu options are as follows:


    Add

    Add the chosen interval to the base frequency.

    Remove

    Removes the chosen interval from the base frequency.

    Delete

    Deletes the specified user-defined interval from the lists.

    Custom...

    Adds a user-defined interval to the list.


    Program Control

    Entrainment Frequency

    This slider controls the difference in frequencies between the right and left stereo channels.

    Frequency

    This slider controls the base frequency. The actual frequency is of little importance to entrainment, as it is the entrainment frequency (the difference between the left and right stereo outputs) which matters.

    Amplitude

    This slider controls the amplitude (volume) of the generated sine waves.

    Pink Noise

    This slider controls the amplitude (volume) of the generated pink noise. Pink noise is included as an audio distraction, something upon which to focus while the generated sine waves do their job.

    Pink Noise Options

    This button opens the pink noise options dialog.

    Pink Noise Options

    From this dialog, the user can control whether the pink noise pans, the panning amplitude (left/right) and the multiple used with the entrainment frequency to determine how fast panning occurs.


    The File Menu

    The file menu options are as follows:


    Start

    Starts Entrainer with the specified settings.

    Stop

    Halts Entrainer execution.

    Load Entrainer Program

    Loads the specified Entrainer program for execution.

    Clear Entrainer Program

    Clears the loaded Entrainer program from memory.

    Edit Entrainer Program

    Edits the specified Entrainer program.

    New Entrainer Program

    Opens the Entrainer Program Editor, allowing the user to create a new Entrainer program.

    Random Background

    When selected a random image from the css directory is chosen as EntrainerFX's background.

    Exit

    Exits the program after a confirmation dialog.

    The Help Menu

    This menu displays the 'About' dialog, an entrainment introduction and the LICENSE.txt file included with the installation. Access to local and web documentation for EntrainerFX is available and in version 5.1.0 a checkbox item exists which allows enabling/disabling of the splash screen on startup.

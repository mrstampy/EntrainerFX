Neural Network Power Signal Analysis for ESP Devices (New in 5.0.0 Final)


The purpose of the Neuralizer panel is to provide visual feedback of the conscious state of the user of EntrainerFX through the use of various EEG hardware devices via the ESP (version 2.1) library. The intent is for the user to either set EntrainerFX at a constant entrainment frequency or to run an EntrainerFX program, process the raw EEG signal and have the result sent through the Neuroph neural network to provide the visual feedback. The Neuralizer network consists of 144 output neurons from 40 power raw signal input neurons corresponding to the EntrainerFX's entrainment frequency range. In order to use the Neuralizer the neural network must first be trained to recognize specific conscious states.

EntrainerFX 5.0.0 includes three ESP implementations; ESP ThinkGear for ThinkGear devices, ESP Nia for the OCZ Nia and ESP OpenBCI (experimental) for OpenBCI hardware.

Choosing an ESP Device

Once the device has been chosen signal acquisition can be started and stopped from the menu. And while the raw signal itself can be used as input into the Neuralizer panel, the raw signal can be processed using the digital signal processing utilities included in the ESP library prior to input into the Neuralizer panel. These utilities are controlled via the ESP Lab interface using the ESP Lab JavaFX GUI. The Lab settings can be saved and loaded as required from the menu. By default the Lab settings are saved in the 'esp.lab' directory.

At a minimum an FFT algorithm should be applied to the raw signal as this transforms the signal from the time domain to the power domain (measuring the relative power of the signal at the various frequencies). Further DSP theory is left as an exercise for the reader.

Developer Information

EntrainerFX uses dynamic class loading for the various ESP devices and controls the devices through the the ESP library interfaces and superclasses. ESP device implementations can be added to EntrainerFX by dropping the implementation into the 'esp' subdirectory. External configuration files (ie: ESP-OpenBCI properties) are placed into the 'esp.config' directory. External implementation dependencies must be specified on the command line (ie: java -cp /path/to/dependency.jar -jar EntrainerFX.jar). With external dependencies EntrainerFX requires a restart, however with no external dependencies the ESP implementation will be picked up without restarting.

Neuralizer Controls

Control of the Neuralizer is accomplished via the mouse and keyboard when the Neuralizer TitledPane is expanded.

Neuralizer Controls (always available)

  • Right click - save current network
  • Left clicking a neuron selects / deselects the neuron
  • Click-drag-click (x direction) controls panel opacity (left to right = opaque, right to left = see thru)

ESP Device Disconnected

  • Shift + Right click - load network

ESP Device Connected

  • Double click (Left) - start / stop recording for network training
  • Shift + Left click - randomize network weightings

Selecting Neurons


Any number or pattern of neurons can be selected. The selected neurons are used as bias neurons when the network is being trained and provide visual queues for the user.

Network Save

Right clicking will bring up the network save dialog.


Network Load


Shift + right click will bring up the network load dialog when disconnected from ThinkGear devices. When a network is saved two files are written, one with the name specified and one with a '.selections' extension. The named file is the saved neural network while '.selections' is a list of selected neurons and is loaded when the network is loaded. Once loaded the neurons will briefly change color to indicate a successful network load.

Connected


After successful connection with the ThinkGear device the neurons change colour and start responding to the power signal from the device. In preparation for neural network training EntrainerFX can be used to entrain one's brainwaves to a desired entrainment frequency state until ready to train the neural network.

Training


Double clicking changes the neuron colour and starts the training process. Recording should be allowed to span at least several seconds to several minutes while EntrainerFX should be used to keep one's conscious state steady. Double clicking again halts the recording and the values are then used to train the neural network. When the selected neurons are fully opaque then the network can be considered trained.

Shift-clicking will randomize the network weightings which may be necessary for network training. The color of the neurons will change briefly to indicate successful randomization.

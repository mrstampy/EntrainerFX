Recording

Entrainer provides the ability to save the audio output from Entrainer programs to WAV files. Using your favourite audio conversion tool (ie. iTunes) you can create mp3's from the generated WAV files, for use on your personal media device. This feature allows the user to use entrainment programs without using a computer.

To record, click the 'Record' icon. The following dialog appears:

WAV File Chooser

Select or type the name of the .wav file to which you wish to save Entrainer's audio output. By default, the WAV file will be saved to the 'wav' subdirectory of the Entrainer install.

Clicking 'Ok' will set up Entrainer to record to the specified WAV file.

Once ready, click the 'Play' icon to begin recording. Pressing the 'Stop' icon will halt recording.

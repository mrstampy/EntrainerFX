Choosing 'Load Entrainer Program' from the file menu brings up the following selection dialog:

Entrainer Program Chooser

Select the Entrainer Program (an xml file) to load. By default the dialog opens in the 'xml' subdirectory of the Entrainer install.


Once selected the file is loaded into the program, and the sound controls become disabled. Choosing 'Clear Entrainer Program' from the file menu will clear the XML from memory and re-enable the controls.

The Entrainer program can now be paused. Simply click the now-enabled pause button to pause/resume the program.

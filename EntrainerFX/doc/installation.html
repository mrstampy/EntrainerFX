The Entrainer installation is an executable jar file created by IzPack, named EntrainerFX[x]Setup.jar, where '[x]' is the release identifier, and can be downloaded from the Entrainer SourceForge project page. Double clicking this file will start the following installation process.

The Introduction Screen

Click 'Next' to continue.

The Notes Screen

This is the contents of the README.txt file included with the installation. It contains information which may be valuable should any problems be encountered. When ready click 'Next' to continue.

The License Screen

This is the contents of the LICENSE.txt file included with the installation. When ready click 'Next' to continue.

The Installation Directory Screen

Choose your installation directory. The default is whereever you've opened up EntrainerRelease[x]Setup.jar; change to any directory of your preference with one caveat: note that the latest Java 7 releases have a bug and the directories where EntrainerFX is installed cannot have spaces in the names. When selected click 'Next' to continue.

The Install Packs Screen

You can optionally choose not to install the JavaDoc (code documentation) and not to install the Java source, as these are not necessary components to run Entrainer. When you've chosen the components to install click 'Next'.

The Progress Screen

When finished click 'Next'.

Installation Complete Screen

Click 'Done' to complete the install of Entrainer.
